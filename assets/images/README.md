# assets/images/

Put real photos here (or upload them straight through the Studio
Dashboard, which stores them for you automatically).

Suggested files:
- founders.jpg     → photo of Ankit & Mohit for the About section
- gallery1.png, gallery2.png, ...  → gallery stills

If you place files here, reference them with a relative path, e.g.:
  <img src="assets/images/founders.jpg" alt="Ankit & Mohit Chouhan">

Or simpler: open the site → Studio Login → Photos tab → paste an
image URL or upload a file directly, no code editing needed.
