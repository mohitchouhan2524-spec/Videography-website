# assets/videos/

Put real video files here if you want to self-host them instead of
using YouTube/Vimeo embeds or the Studio Dashboard's URL/upload fields.

Suggested files (referenced nowhere by default — the site currently
uses the Studio Dashboard to add hero/portfolio video URLs instead):
- hero.mp4        → a short, muted, looping background reel for the hero section
- wedding1.mp4     → a sample wedding film

Tips:
- Keep hero background videos short (10–20s loop) and under ~10MB,
  muted, no audio track if possible, for fast loading.
- For anything longer than a quick loop, prefer uploading to
  YouTube/Vimeo (unlisted is fine) and pasting the embed URL instead
  of self-hosting — it's much lighter on your site's load time.
